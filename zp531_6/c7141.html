    How to adjust the resulotion?
    Step 1: Install Raspbian official image
    a) Download the latest image from the official download
    b) Install the system according to the official tutorial steps

    Step 2: modify the ‘config.txt’
    After the programming of Step 1 is completed, open the config.txt file of TF card root directory and add the following code at the end of the file, save and eject Micro SD Card safety:
    hdmi_force_edid_audio=1
    max_usb_current=1
    hdmi_force_hotplug=1
    config_hdmi_boost=7
    hdmi_group=2
    hdmi_mode=1
    hdmi_mode=87
    hdmi_drive=1
    hdmi_cvt 800 480 60 6 0 0 0
