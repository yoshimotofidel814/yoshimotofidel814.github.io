How use mechanical watch?

- 1. When you first wear, turn the pocket watch button (clockwise) until it is full (15-18 laps).
- 2. Softly pull the crown up, then turn the crown to adjust the time. Press the crown to start up the pocket watch.
- 3. If not used and stop for a long time, please gently swing several times, winding a few laps. At this point it will work properly.
- 4. If you perceive the wind is tight, please promptly cease rotating, so as not to harm the movement.
